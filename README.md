# GPA Calculator  
The program takes `4` input from the user and then calculates the average number. Grade calculated according to this table
| Result | Grade  |
|-------|--------|
|  0-40  |   F    |
| 41-60 |   D    |
| 61-70 |   C    |
| 71-80 |   B    |
| 81-90 |   A    |
| 91-100|   A+   |
